# 常用正则表达式



#### 1、身份证

```java
^[1-9]\d{5}[1-9]\d{3}((0\d)|(1[0-2]))(([0|1|2]\d)|3[0-1])\d{3}(\d|x|X)$
```

#### 2、军官证

```java
^[0-9]{8}$
```

#### 3、护照

```java
^[a-zA-Z0-9]{5,17}$
```

#### 4、营业执照

```java
^[a-zA-Z0-9]{10,20}$
```

#### 5、驾照

```java
^[1-9]\d{5}[1-9]\d{3}((0\d)|(1[0-2]))(([0|1|2]\d)|3[0-1])\d{3}(\d|x|X)$
```

#### 6、组织机构代码证

```java
^[a-zA-Z0-9]{10,20}$
```

#### 7、台胞证

```java
^([0-9]{8}|[0-9]{10})$
```

#### 8、港澳通行证

```java
^[a-zA-Z0-9]{6,10}$
```
